## CAT WALK

L'objectif de cet exercice est de faire courir un chat grâce à la méthode ```setInterval```.

L’URL de la première image est : [https://raw.githubusercontent.com/Innotech-Academy-Corporation/catwalkimages/main/cat-1.jpg](https://raw.githubusercontent.com/Innotech-Academy-Corporation/catwalkimages/main/cat-1.jpg )

Le nom des 10 images va de cat-1.jpg à cat-10.jpg.

Une fois la dernière image affichée, la première est de nouveau affichée et ainsi de suite.

```html```
```html
<!DOCTYPE html>
<html>
    <head>
    	<link rel="stylesheet" href="style.css">
    </head>
    <body>
    	<img id="cat-run">
    	<script src="script.js"></script>
    </body>
</html>
```

```style.css```
```css
body {
    margin: 0;
    height: 100vh;
    width: 100vw;
    display: flex;
    justify-content: center;
    align-items: center;
}
```

```script.js```
```js
function getCatImage(imgNumber) {
	return 'https://raw.githubusercontent.com/Innotech-Academy-Corporation/catwalkimages/main/cat-1.jpg ';
}

document.querySelector('#cat-run').src = getCatImage(1);
```

Prenez connaissance du code fourni et modifiez-le afin de donner l’impression que le chat court, 
en changeant d’image toutes les 100 ms, par exemple.

Vous devez obligatoirement utiliser la fonction "```getCatImage```" qui est censée renvoyer l’URL de l’image du chat 
en recevant l’index en paramètre (de 1 à 10).
